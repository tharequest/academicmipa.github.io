# academicmipa.github.io

A static academic website for the Faculty of Mathematics and Natural Sciences (MIPA), built using **HTML, CSS, and JavaScript** and deployed via **Vercel**.

## 🌐 Live Demo
👉 https://fmipa-portalacademic-edu.vercel.app/
---

## 📁 Project Structure

- css/
- js/
- images/
- fonts/
- index.html
- statsurat.html
- README.md
---

## ✨ Features

- 📸 **Photo Upload**
  - Supports image upload and display from the `images/` directory
- 🎨 **Custom Styling**
  - Modular CSS for clean and maintainable UI design
- 🧩 **Interactive JavaScript**
  - Client-side logic handled in the `js/` directory
- 🖋️ **Custom Font Integration**
  - Uses **Coinbase Sans** for a modern and professional look

## 🚀 Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/tharequest/academmicmipa.github.io.git
2. Navigate to the project directory:
   ```bash
   cd academmicmipa.github.io

3. Open index.html directly in your browser
or use a local development server (e.g., VS Code Live Server).
